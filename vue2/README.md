# 响应式

## 依赖收集

依赖指的是使用到某个数据的组件、函数。是数据的使用者、依赖者。  
依赖收集指的是收集依赖者。因为数据发生变化时，需要通知给依赖者做出更新、变化。

## 数组响应式

Object.defineProperty 可以劫持数组，但如果每个属性都劫持，数组越长，性能就越差。具体表现在几个方面：

- 初始化性能差，大型数组遍历所有索引，加 getter/setter，耗时长
- 动态维护成本高，新增/删除需要增加删除 getter/setter
- 高频操作更新冗余，批量修改/删除、sort 排序，触发大量无效的更新
- 内存占用大，大型数组每个元素都要收集依赖，浪费内存

所以 vue2 在响应式处理数组时，是重写了数组的原型对象，将 7 个方法重写，在方法里触发更新。
